Verilog Xilinx Vivado FPGA IR Sensor
Project / 01 · Completed

IR Proximity Sensor with Distance Measurement

Designed and implemented an infrared proximity sensor system on an FPGA board using Verilog HDL and Xilinx Vivado. The system detects nearby objects and estimates distance by timing IR pulse reflections, with real-time output. My first actual hardware project — I learned the hard way that simulation and real hardware behave very differently.

Honest reflection
Debugging on real hardware was tougher than expected. Spent a lot of time on timing constraints and sensor threshold calibration. Got it working after plenty of iteration — and learned more from the failures than the successes.
